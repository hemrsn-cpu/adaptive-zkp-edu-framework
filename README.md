This project proposes an **Adaptive Zero-Knowledge Proof (ZKP) Framework** designed to balance the trade-off between **verification cost** and **data privacy** for educational credentials.

In the era of PDPA and GDPR, verifying academic qualifications shouldn't require exposing sensitive personal data. This framework allows institutions to verify credentials (e.g., "Has a Bachelor's degree") without revealing unnecessary details (e.g., grades, exact graduation date), while dynamically adjusting the computational cost based on the verification context.
